Shakouri Calculator - Android Studio project (skeleton)

What this is:
- A ready-to-import Android Studio project (Kotlin) containing:
  * Modern Material-styled calculator UI
  * Separate screen for GCD (ب.م.م) and LCM (ک.م.م)
  * App name: "shakouri calculator"

How to build an APK:
1. Open Android Studio (Chipmunk or newer recommended).
2. Choose "Open" and select the folder 'ShakouriCalculator' (this folder).
3. Let Android Studio download Gradle and sync.
4. Build > Build Bundle(s) / APK(s) > Build APK(s).
5. The generated APK will be in app/build/outputs/apk/debug/app-debug.apk

Notes:
- This is a skeleton project and uses modern dependencies. Make sure your Android SDK is installed.
- If you want, I can modify colors, icon, or add an app icon pack.
